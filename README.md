## Local deployment

### Dependencies
* Node v12.16.3 and higher
* NPM v6.14.4 and higher
* React v17.0.1
* React-dom": "^17.0.1",
* Classnames v2.2.6
* Prop-types v15.7.2

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`
